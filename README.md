# 💅 Studio de Sobrancelhas - Website

Projeto de site desenvolvido para apresentação de serviços estéticos, com foco em design de sobrancelhas, cílios e atendimento personalizado.

---

## 📌 Sobre o Projeto

Este site foi criado com o objetivo de divulgar serviços, facilitar o contato com clientes e apresentar os procedimentos de forma visual e atrativa.

O layout foi pensado para transmitir elegância, profissionalismo e modernidade.

---

## 🚀 Funcionalidades

- Apresentação dos serviços (Henna, Design Masculino, Cílios)
- Botão de agendamento direto via WhatsApp
- Layout moderno com foco visual
- Efeitos de hover nas imagens
- Estrutura organizada em seções

---

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3  
- Responsividade (em ajuste)

---

## 📱 Responsividade

O site atualmente está otimizado para desktop.

Melhorias de responsividade para dispositivos móveis serão implementadas em etapas futuras, garantindo adaptação sem comprometer a estética do layout.
